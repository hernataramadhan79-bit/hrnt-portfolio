# Hernata Ramadhan - Personal Portfolio

<div align="center">
  <h3>Modern, Interactive & Futuristic Portfolio</h3>
</div>

Welcome to the repository of my personal portfolio website! This project is designed as a highly interactive, futuristic, and premium showcase of my professional experience, skills, and open-source contributions. It blends Cyberpunk-inspired neon aesthetics with modern Liquid Glass (iOS 16 style) visuals.

## ✨ Features

- **Interactive UI & Fluid Animations**: Powered by Framer Motion for smooth scroll effects, hover states, and dynamic page transitions.
- **Live Performance Metrics**: Real-time integrations with the **GitHub API** (to show commits, stars, top repositories) and **WakaTime** (to display accurate coding hours and language statistics).
- **Responsive Layout**: Designed to look stunning and scale perfectly across mobile, tablet, and desktop devices.
- **Projects Showcase**: A dedicated section to display web applications, open-source work, and freelance projects.
- **Professional Experience Matrix**: A structured timeline highlighting roles, technologies used, and core achievements.
- **Digital Certificates**: A gallery of technical and professional certifications.
- **Dynamic Contact Form**: Handled via Web3Forms for seamless client and collaborator outreach.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Library**: [React](https://reactjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Run Locally

Want to run this project in your own local environment? Follow these steps:

**Prerequisites:** Node.js (v18 or newer recommended).

1. Clone the repository and navigate into the directory:
   ```bash
   git clone <your-repo-url>
   cd hrnt-portfolio
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Environment Variables (if needed for APIs like Wakatime/Github):
   Create a `.env.local` file in the root directory and add any required tokens.

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit:
   ```
   http://localhost:3000
   ```
